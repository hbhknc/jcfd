# 🚒 Fire Rescue Incident Dashboard

An interactive, real-time web application built with Python and Streamlit to track, visualize, and analyze fire department incident logs from 2020 to 2025. 

## 🌟 Features

* **Interactive Slicers:** Filter the entire dashboard by Year and Call Category in real-time.
* **Dynamic KPI Cards:** Quick, at-a-glance metrics showing total incidents, most frequent call types, mutual aid given, and the date of the last recorded incident.
* **Temporal Analysis:** A spline chart detailing the volume of calls over time (grouped by month) to identify busy seasons or historical trends.
* **Categorical Breakdown:** A horizontal bar chart ranking the frequency of different incident types (e.g., MVCs, Structure Fires, Brush Fires).
* **Raw Data Matrix:** A clean, searchable data table displaying the underlying incident records and dispatcher notes for the selected filters.

## 🛠️ Tech Stack

* **Framework:** [Streamlit](https://streamlit.io/)
* **Data Manipulation:** [Pandas](https://pandas.pydata.org/)
* **Data Visualization:** [Plotly Express](https://plotly.com/python/plotly-express/)
